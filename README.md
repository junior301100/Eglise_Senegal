# Église Catholique – Sénégal

Site web pour trouver les paroisses, voir les horaires des messes et lire les prières du jour.

## Fonctionnalités

- 🏛️ **8 paroisses** avec horaires et contacts
- 🗺️ **Carte interactive** pour localiser les églises
- 📖 **Lectures liturgiques** du jour
- 🙏 **Saint du jour** avec description
- 📱 **Responsive** sur mobile et desktop

## Installation

1. Copier les fichiers dans un serveur web (Apache, Nginx, XAMPP)
2. Accéder au site via le navigateur
3. C'est prêt ! ✅

## Structure

```
paroisse/
├── index.html          # Page principale
├── css/style.css       # Styles
├── js/
│   ├── data.js         # Données paroisses et saints
│   ├── app.js          # Application principale
│   └── pages/          # Pages du site
└── README.md           # Ce fichier
```

## Pages

- **Accueil** : Saint du jour et prochaines fêtes
- **Paroisses** : Liste et recherche des églises
- **Détail** : Informations complètes d'une paroisse
- **Liturgie** : Lectures et prières du jour
